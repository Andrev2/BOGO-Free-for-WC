Automatically adds one or more free products to the cart when a specific product or category is purchased. Perfect for promotions and bundle deals!

== Description ==
BOGO Free for WC is a plugin that allows you to automatically add free products to the customer's cart when specific products or categories are purchased. You can set target products and categories, and specify which products will be added for free.

== Features ==
* Add one or more free products to the cart when target products are in the cart.
* Target specific products or product categories.
* Automatically set the price of free products to zero.
* Seamlessly integrates with WooCommerce.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/bogo-free-for-wc` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to "BOGO Free for WC" in the admin menu to configure the settings.

== Frequently Asked Questions ==

= Can I specify multiple free products? =
Yes, you can specify multiple product IDs separated by commas.

= What happens if I remove a target product from the cart? =
The free products will also be removed from the cart if the target products are no longer present.
